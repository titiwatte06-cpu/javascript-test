ในเกมนี้มีวิธีและ step การวางแผนดังต่อไปนี้คือ
1.การสร้างตารางออกมาก่อน
2.การใส่ตัวผู้เล่นลงไป และ ใส่ตัวจุดหมาย
3.ใส่ค่าหลุมลงไปและใส่เงื่อนไข if-else กรณีที่เราเดินแบบตกหลุม

ในตอนแรกเราจะแบ่งโค้ดออกเป็น 2 ส่วน คือส่วนที่เป็นการสร้างตารางและส่วนที่เป็นการเล่นเกม โดยในส่วนที่เป็นการสร้างตาราง ก็จะใช้วิธีการ for loop เพื่อวนค่าไปเรื่อยๆและทำให้เกิดตารางขึ้นมา 

ในส่วนของการเดินก็จะใช้วิธีการ array method เพื่อ push ค่า array ไปทำให้เกิดการเปลี่ยนแปลงในตารางขึ้นมาและในตารางก็จะมีการวางเงื่อนไขก็คือถ้า เดินไปเหยียบหลุมก็จะให้เกมหยุดลง โดยหัวใจสำคัญของการเช็คค่าในเงื่อนไข if-else คือการ ดูว่าช่องที่กดมันอยู่ใน index อะไรและก็เข้าลูปตามเงื่อนไขนั้น

ในส่วนแรกของ code เราจะทำการเก็บค่าที่จำเป็นใน class field เพื่อให้เอาไปใช้ต่อได้ในเก็บ เช่น

ค่าที่หมายถึงตัวตาราง
-หัวใจหลักของการสร้างตารางแบบ array 2 มิติคือการสร้าง for loop ซ้อนกันขึ้นมาโดยอันแรกซ้อนแบบแถว อันที่สองซ้อนเพื่อหมายถึง column โดยสาเหตุทีทำแบบนี้เพื่อให้ตารางของเราสามารถสร้างค่าขึ้นมาได้
